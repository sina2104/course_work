## Чат-бот погоды
Чтобы обучить чат-бот:
```shell
   rasa train
```
Чтобы запустить чат-бот, сначала запустите файл actions в отдельном терминале следующим образом:
```shell
   rasa run actions
```
Теперь можно пользоваться чат-ботом командой:
```shell
   rasa shell
```
